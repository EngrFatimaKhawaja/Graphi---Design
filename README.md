# Logo Design
The two main logos which I have designed using Canva.
## Bloom Aura Studio
Bloom Aura Studio is a beauty and self-care brand that offers skincare, makeup services, and wellness treatments.
The brand represents elegance, calmness, confidence, and natural beauty.
## EcoBrew Café
EcoBrew Café is a modern, eco-friendly coffee shop that focuses on organic, sustainable, and environmentally friendly products.
The café promotes green living and reduces plastic waste by using biodegradable cups and packaging.
